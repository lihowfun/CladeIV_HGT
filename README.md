Clade IV nematode Horizontal gene transfer events

This repository contains the data used in the analysis of horizontal gene transfer (HGT) events in the last common ancestor of clade IV nematodes, as published in Molecular Ecology (The Aphelenchoides genomes reveal substantial horizontal gene transfers in the last common ancestor of free-living and major plant parasitic nematodes)

Data

The data for this analysis consists of the following files:

alignment.fasta: A multiple sequence alignment of genes used in the HGT analysis.
HGT_families.txt: A list of HGT families identified in the analysis.
phylogeny.pdf: A phylogenetic tree showing the relationships among the nematode species used in the analysis.
