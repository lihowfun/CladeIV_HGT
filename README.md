# Clade IV nematode Horizontal gene transfer events

This repository contains data related to the analysis of 161 horizontal gene transfer (HGT) events in the last common ancestor of clade IV nematodes, as published in [The Aphelenchoides genomes reveal substantial horizontal gene transfers in the last common ancestor of free-living and major plant parasitic nematodes]

## Data

The 161 HGT families consists of the following files:

1. all_fasta: HGT copies of nematodes and their closely related genes in FASTA format.
2. all_alignment: A multiple sequence alignment of the HGT families.
3. all_tree: A phylogenetic tree showing the relationships among the nematode species containing HGT copies and their donor species.
